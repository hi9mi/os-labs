# Лабораторные работы по ОС

В этом репозитории собраны лабораторные работы по дисциплине «Операционные системы».

## Состав

- `lab1` - batch-скрипты для работы с файлами и аргументами командной строки.
- `lab4` - программа на Python для создания и рекурсивного копирования ветки реестра Windows.
- `lab5` - графическое приложение на Python для просмотра процессов и модулей в Windows.

## Как пользоваться

У каждой лабораторной работы есть свой `README.md` с описанием и инструкцией по запуску:

- lab1/README.md
- lab4/README.md
- lab5/README.md

## Примечание

Работы ориентированы на Windows, так как используют:

- `bat`-скрипты;
- реестр Windows;
- системные процессы и модули Windows.
