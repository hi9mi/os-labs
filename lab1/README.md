# Лабораторная работа 1

Набор `bat`-скриптов для работы в командной строке Windows.

## Состав работы

- `task1.bat` - рекурсивно ищет файлы, имена которых начинаются с `f`, а расширение начинается с `p`.
- `task2.bat` - рекурсивно выводит файлы с расширением из одного символа.
- `task3.bat` - рекурсивно выводит файлы и папки, чьи имена начинаются с `a`, а также с `ba`.
- `compile.bat` - формирует команду запуска компилятора `tcc`, разделяя входные параметры на файлы и ключи.

## Требования

- Windows;
- `cmd.exe`;
- для `compile.bat` нужен установленный `tcc`, если планируется реальная компиляция.

## Запуск

Откройте `cmd` или PowerShell в папке `lab1`.

### task1.bat

Пример:

```cmd
task1.bat C:\
```

Скрипт ищет пути вида:

- `f*.p*`

## task2.bat

Пример:

```cmd
task2.bat C:\
```

Скрипт ищет файлы, у которых расширение состоит из одной буквы, например:

- `file.c`
- `notes.h`

## task3.bat

Пример:

```cmd
task3.bat C:\
```

Скрипт выводит объекты, имя которых:

- начинается с `a`
- начинается с `ba`

## compile.bat

Пример:

```cmd
compile.bat main.c /O2 test.c /Wall
```

Скрипт разбирает аргументы и выводит итоговую команду в виде:

```cmd
tcc main.c test.c /O2 /Wall
```

## Примечание

Во всех скриптах, если путь или аргументы не переданы, выводится подсказка с примером запуска.
