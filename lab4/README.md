# Лабораторная работа 4

Программа на Python для создания тестовых данных в реестре Windows и рекурсивного копирования одной ветки реестра в другую.

## Что делает программа

- создаёт ветку `HKEY_CURRENT_USER\Software\Lab4Source`;
- добавляет в неё несколько строковых и числовых значений;
- создаёт вложенные подразделы `Settings` и `Profile`;
- копирует всю ветку `Lab4Source` в `HKEY_CURRENT_USER\Software\Lab4Copy`;
- копирует значения типов `REG_SZ` и `REG_DWORD`.

## Требования

- Windows;
- Python 3.x;
- доступ к реестру текущего пользователя.

## Запуск

Откройте терминал в папке `lab4` и выполните:

```powershell
python main.py
```

## Как работает программа

1. Функция `create_test_data()` создаёт тестовые разделы и значения в реестре.
2. Функция `copy_values()` копирует значения из одного раздела в другой.
3. Функция `copy_key_recursive()` рекурсивно копирует раздел и все его подразделы.
4. В `main()` сначала создаются тестовые данные, затем выполняется копирование.

## Какие ветки используются

Исходная ветка:

```text
HKEY_CURRENT_USER\Software\Lab4Source
```

Ветка назначения:

```text
HKEY_CURRENT_USER\Software\Lab4Copy
```

## Ограничения

- программа рассчитана на Windows, потому что использует модуль `winreg`;
- копируются только значения типов `REG_SZ` и `REG_DWORD`;
- если раздел назначения уже существует, его значения могут быть перезаписаны.
